# KGF Restaurant (Clone of KFC Website)
 
## Overview
---
This project is a clone of the popular KFC (Kentucky Fried Chicken) website, rebranded as KGF (King of Good Food). The objective of this project is to provide a seamless user experience and ensure a fast end-to-end flow for both users and admins.
               

### Tech Stack
___

* Frontend: HTML5, CSS3, JavaScript, React.js, Redux.js

* Backend: Node.js, Express.js, MongoDB
